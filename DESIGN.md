# 母婴记录网站

## 需求
1. 按月组织宝宝的生活记录，包括照片、视频、育儿知识、辅食食谱、早教视频等，并生成时光轴
    * 总结昨日宝宝睡眠、喂养、大便情况
2. 通过百度网盘查看手机上传的宝宝照片、视频
3. 每天搜罗该月龄宝宝的育儿知识、辅食食谱，以及早教视频
    * 可以多搜索微信公众号、小红书、抖音
    * 还有比如丁香医生，崔玉涛医生，卓正医疗等知识来源
4. 宝宝生活记录：喂奶、睡眠、换尿布时间精准记录，亲喂、瓶喂、辅食情况也能详细记载，甚至还可以自主编辑条目，培养宝宝规律作息的神器
5. 生长曲线评估：记录身高、体重、头围数据，结合世卫组织百分位曲线，直观了解宝宝成长状态
6. 自动算月龄，到疫苗/体检时间自动提醒
7. 便捷操作：表格式记录，操作简单，一目了然
    * 也需要提供 API ，方便 OpenCLAW 或者 Claude Code 来操作
