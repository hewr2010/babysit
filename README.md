# babysit - 宝宝成长管家

简洁美观的单页面宝宝成长记录应用。

## 功能

- 👶 **宝宝信息** - 姓名、生日、月龄自动计算
- 📅 **日期切换** - 按日期查看和记录
- 📝 **生活记录** - 喂奶、睡眠、换尿布
- 📸 **每日相册** - 百度网盘照片按日期展示，显示拍摄时间
- 💉 **疫苗提醒** - 根据月龄自动提醒疫苗和体检
- 📊 **热力图** - 图标化显示每日记录，一目了然

## 安装

```bash
# 安装
uv pip install -e .

# 配置百度网盘
bypy info
```

## 使用

```bash
python babysit/app.py
```

访问 http://localhost:8080

## 数据存储

所有数据存储在项目目录内：
- `babysit/data/babysit.db` - SQLite 数据库
- `babysit/data/cache/` - 百度网盘文件列表缓存

注：照片和视频直接使用百度网盘直链，无需本地存储。

## 配置

修改百度网盘路径：
```bash
export BABYSIT_PATH="/你的相册路径"
```

## 主题

应用采用温馨的公主粉配色，更适合记录女宝宝的成长。主要颜色：
- 主色：粉红色 (#ec4899)
- 背景：渐变粉色
- 图标：使用emoji表情，更生动可爱

## 界面

- 顶部：宝宝信息 + 月龄
- 昨日总结：显示昨天的睡眠/喂奶/换尿布统计
- 热力图：按月展示所有记录，图标化显示
- 生长记录：身高/体重/头围曲线
- 疫苗提醒：根据月龄显示近期疫苗和体检
- 相册：按月展示百度网盘照片，支持查看原图
- 快捷按钮：底部固定，快速记录喂奶/睡眠/换尿布/生长
